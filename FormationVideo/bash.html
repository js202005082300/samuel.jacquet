	## Commandes diverses
	- [ ] Environnement Linux
	* ctrl+H 				(afficher les fichiers cachés)
	* lsb_release -a			(Version distribution)
	* sudo -i				(Passer en ROOT)
	* ctrl+D				(Sortir du mode ROOT)
	* su					(Passer en Super-Utilisateur)
	* javac 				(tester l'installation du JDK)
	* java -version 			(tester la version)
	* g++ --version 			(vérifier la version)
	* python3
	
	- [ ] rm
	* rm 				(Supprimer un fichier)
	* rm -r 			(Supprimer un dossier)
	
	- [ ] imagemagick
	* import -window root Images/screenshot.png
	* import Bureau/Partage/snap.png 			(screenshot d'une selection)
	* import -window root -pause # Bureau/Partage/fileName.png 	(# = compte à rebours en seconde)
	
	- [ ] icoutils
	* icon.sh                           	(Bash shell - ksh)
	* rm *.ico                          	(supprimer tout les .ico)
	
	- [ ] youtube-dl
	* youtube-dl -x --audio-format mp3 url_de_la_vidéo	(extension mp3)
	* youtube-dl -F 22 url_de_la_vidéo			(extension mp4 1280x720 [best])
	
	- [ ] Vim
	* i                                 	(insertion)
	* :w                                	(sauver)
	* :q                                	(quitter)
	* :wq (ou :x)                       	(sauver et quitter)
	* :q!                               	(quitter et annuler)
	* /texte (ou ?texte)                	(rechercher texte)
	* n                                 	(occurence précédente)
	* N                                 	(occurence suivante)
	* :%s/ancien-mot/nouveau-mot/g	(remplacer ancien-mot par nouveau-mot)
