cours # -  Apache & PHP


Linux


installer serveur Apache

sudo apt-get install apache2        (installation d'Apache)
sudo systemctl disable apache2    (empecher le service de se lancer au démarrage de l'OS)
sudo systemctl start apache2      (démarrer le service)
sudo systemctl stop apache2       (couper le service)
sudo systemctl restart apache2    (redémarrer)

Editer nos pages web

ls -l /var/www/html                 (tester nos droits sur ce répertoire)
sudo chmod 755 -R /var/www/html             (accorder l'accès)
sudo chown $USER:$USER -R /var/www/html     (devenir propriétaire)

Installer l'interpreteur PHP

sudo apt-get install php libapache2-mod-php (installation de PHP et du module d'Apache)

Editer /etc/apache2/mods-enabled/php7.4.conf

sudo vim /etc/apache2/mods-available/php7.4.conf    (éditeur, lien version 7.4)
php_value display_errors on                         (en fin de fichier)

Editer /etc/apache2/apache2.conf

sudo vim /etc/apache2/apache2.conf
			
	Options Indexes FollowSymLinks
	AllowOverride All	(None devient All, permettre .htaccess)			
	Require all granted			

AccessFileName  .htaccess	(par défaut si tout va bien)

Editer /etc/php/7.4/apache2/php.ini

sudo vim /etc/php/7.4/apache2/php.ini
error_reporting = E_ALL				(ligne 465)
display_errors = On					(ligne 482)
display_startup_errors = On				(ligne 493)
html_errors = On					(ligne 552)
default_charset = "UTF-8"				(ligne 713)

Editer nos pages PHP

sudo vim /var/www/html/index.php

Test
sudo systemctl start apache2
localhost/
localhost/monsite/					(/var/www/html/monsite/)
sudo systemctl stop apache2
