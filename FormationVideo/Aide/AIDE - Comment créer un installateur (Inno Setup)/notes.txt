AIDE - Comment créer un installateur (Inno Setup)
https://www.youtube.com/watch?v=ormsdIk_Uhw
Plusieurs types d'installateur : payant, open source, gratuit, etc.
Inno setup : gratuit, écrit en pascal si on s'interesse au script.
Télécharger sur https://www.jrsoftware.org/isinfo.php
Récupérer l'exe : Downloads -> Stable Release -> Random site.
Note : passer un png dans un convertisseur .ico en ligne.
Démarrer Inno Setup Compiler :
1.	Cocher "Create a new script file using the Script Wizard" -> Next
2.	Welcome … -> Next
3.	Remplisser 
a.	"Application Name" : …
b.	"Application Version" (suivant notre système de version) : 1.0
c.	"Application Publisher" (nom d'entreprise) : …
d.	"Application Website" (info sur le jeu, contact, …) : https://... 
-> Next
4.	Laisser par défaut : "Application destination base folder" et laissé cocher "Allow user to change the application folder" et éventuellement adapter "Application folder name" -> Next.
5.	Spécifier le chemin de notre exécutable : "Application main executable file" et spécifier tous les fichiers du répertoire : "Add file(s)…" (par exemple, le fichier lu avec fopen) -> Next.
6.	Spécifier les raccourcis : "Application Shortcuts"  "Create an Internet Shortcut in the start menu folder" si on a renseigné un site au début de la configuration OU "Create an Uninstall Shortcut in the start menu folder" si on veut créer un désinstalleur dans le menu démarrer -> Next.
Note : éviter d'abuser des raccourcis bureau.
7.	Application documentation, c'est comment on veut distribuer notre programme (payant, gratuit, open source, etc.) :
a.	S'il y a une licence (retrouver les fichiers texte en ligne à incorporer par exemple le fichier de licence GPL trouvé sur les sites dédiés).
b.	Ajouter des informations (épileptiques, daltoniens, etc.) .txt : "Information file shown before installation".
c.	Idem après l'installation (configurations, utilisations, etc.) : " Information file shown after installation".
-> Next.
8.	Setup Install Mode : Laisser le mode administrateur : "Administrative Install Mode (Install for all users)".
9.	Setup Languages : Sélectionner les langues pour les utilisateurs : "Deselct all" et sélectionner ce dont a besoin -> Next.
10.	Compiler Settings : 
a.	"Custom Compiler output folder" = emplacement de notre exécutable à la fin.
b.	"Compiler Output base filename" = nom de l'installateur (ex : installation).
c.	"Custom setup icon file" = fichier .ico.
d.	"Setup password" : Non nécessaire. L'installation décompresse les fichiers dans program files et modifie notre base des registres. On installe rarement un mot de passe -> Next.
e.	Inno setup Preprocesseur : Next.
f.	Finish.
g.	Inno Setup Compiler : générer l'exécutable avec le langage Pascal (script d'installation) -> Non.
